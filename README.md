# 密码保护的网页项目

这是一个使用 URL Hash 密码保护的网页项目。

## 🌐 访问网站

**网站地址：** https://shiying-zhang.github.io/webpage/

## 演示密码

**密码：** `demo123`

## 工作原理

1. 用户输入密码
2. 密码被 SHA256 hash
3. 跳转到以 hash 命名的文件夹
4. 该文件夹包含受保护的内容

## 如何设置你自己的密码和内容

### 步骤 1：确定密码
想好一个强密码（例如：`mySecretPassword123`）

### 步骤 2：获取密码的 Hash
1. 在浏览器访问你的网站
2. 输入你想要的密码
3. 网址会变成：`https://你的用户名.github.io/webpage/#一串hash值`
4. 复制 `#` 后面的那串 hash 值

### 步骤 3：创建内容文件夹
1. 在本地项目目录创建一个以该 hash 命名的文件夹
2. 在文件夹里放入你的 `index.html` 和其他文件
3. 提交并推送到 GitHub

### 步骤 4：删除演示文件夹
删除这个演示用的文件夹：
```
d3ad9315b7be5dd53b31a273b3b3aba5defe700808305aa16a3062b76658a791/
```

## 注意事项

- 不要在公开场合分享包含 hash 的完整 URL
- 定期更改密码以提高安全性
- 这种保护适合一般隐私需求，不适合高度敏感信息
